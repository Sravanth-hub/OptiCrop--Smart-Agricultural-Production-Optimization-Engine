# 🌱 OptiCrop - Smart Agricultural Production Optimization Engine

An AI-powered Machine Learning web application that recommends the most suitable crop using soil nutrients and environmental parameters. The application is built with **Flask** and powered by a **Logistic Regression** model to provide accurate crop recommendations through a simple, interactive, and user-friendly web interface.

## 🚀 Live Demo

🔗 **Website:** https://opticrop-smart-agricultural-production-8bd1.onrender.com

## Clone the repository:

```bash
git clone https://github.com/Sravanth-hub/OptiCrop--Smart-Agricultural-Production-Optimization-Engine.git
cd OptiCrop--Smart-Agricultural-Production-Optimization-Engine
```

## ✨ Features

- 🌱 Predict the most suitable crop using Machine Learning
- 🤖 Logistic Regression-based prediction model

- 🌐 Interactive Flask web application

## 📊 Input Features

The prediction model uses the following agricultural parameters:

- 🌿 Nitrogen (N)
- 🌿 Phosphorous (P)
- 🌿 Potassium (K)
- 🌡️ Temperature
- 💧 Humidity
- ⚗️ pH Value
- 🌧️ Rainfall

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Sravanth-hub/OptiCrop--Smart-Agricultural-Production-Optimization-Engine.git
cd OptiCrop--Smart-Agricultural-Production-Optimization-Engine
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the application

```bash
python app.py
```

### 4. Open in your browser

```
http://127.0.0.1:5000
```
